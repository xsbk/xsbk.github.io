typescript

1、概念

  是JavaScript的一个超集，主要提供类型系统和对ES6的支持
  微软开发的一款开源编程语言
  更像后端JAVA、C#语言，让js可以开发大型项目

2、安装

  npm install -g typescript
  可安装visual studio的typescript插件
  cnpm install -g typescript
  yarn golbal add typescript
    yarn 安装：npm/cnpm install -g yarn

3、转化js 编译ts代码

  tsc name.ts
  vscode自动编译.ts文件
    在文件里创建tsconfig.json文件
      cmd进入文件目录 tsc –init
    配置json文件
      ”outDir”: “./js”, //生成的js目录
    配置vscode
      点击Terminal–>Run tak…–>typescript–>watch 路径/.json文件
    保存ts文件自动编译成js文件。

4、数据类型：（类型注解）

  原始数据类型
    布尔值定义
      let name:boolean = 值;
      使用构造函数创建的布尔对象不是布尔类型 返回的是一个对象
    数值定义
      let name:number = 值;
    字符串定义
      let name:string = 值;
    空值定义
      let name:void =undefined/null;
      function name():void{} //返回值为空
    null/undefined
      let name:null=null;
      let name:undefined;
      是所有其他类型的子类型。可以赋值给其他类型，但是void不行。
    任意值 any
      let name:any=值
      允许赋值任何类型值
      声明未指定类型，默认任意类型（声明时未赋值）
    类型推论
      声明对象未明确指定类型时，且已经赋值了，会根据值推断类型。
    联合类型
      表示取值可以为多种类型中的一种。
      let name:string|number=值; //用 |分割开
      访问不确定联合类型的变量到底是什么类型的属性时，只能访问他们的共有属性
  对象类型
    接口
      赋值时，变量形状与接口需要一致。
        Interface Person ={}
        let name:Person={}
      可选属性：
        proName?:类型 ; 分号结束
      任意属性
        [proName:类型]:类型;
        确定属性和可选属性都是任意属性类型的子集
      只读属性：
        readonly proName :类型;
      封装ajax

      可索引接口 数组、对象约束（不常用）
        数组

      类类型接口 对类的约束 implements（实现接口）
      接口扩展
        接口可以继承接口

    数组的类型
      类型[]

      数组泛型

      接口表示数组
      类数组
        不能用普通数组表示，需要用接口数组表示

        类数组有自己定义的接口 IArguments, NodeList, HTMLCollection 等
      any在数组中
        any[]表示数组中允许出现任意值
    元组类型：
      已知一个数量和类型的数组，各个元素的类型不必相同
      let namespace:[类型,类型…]=[val,val…]
    函数的类型
      函数表达式

      接口表示函数

      可选参数
        proName?:类型;
      默认参数
        proName: string = ‘val’
      剩余参数
        …rest:类型[]表示剩余参数，可变参数
      重载

    类型断言
      语法：
        值 as 类型;
        <类型>值;
      用途：
        将一个联合类型断言为其中一个类型
        将一个父类断言为更加具体的子类
        将任何一个类型断言为any
      双重断言
        as any as 类型;
      类型断言vs类型转换
        类型断言不会影响到变量的类型
      类型断言vs类型声明
      类型断言vs泛型
    声明文件
      新语法索引
        declare var 全局变量声明
        declare function 全局函数声明
        declare class 声明全局类
        declare enum 声明全局枚举
        declare namespace 声明全局对象
        export 导出
        export namespace 导出对象
        export default ES6默认导出
        export = commonjs 导出模块
        declare global扩展全局变量
        declare module 扩展模块
      声明语句
    枚举：enum
      enum name{name1=val1,name2=val2…} 值为整数类型，默认是前一个值加一
      var key:name=name.name1;//key是val1
      enum name{name1,name2…}
      var key:name=name.name1;//key是0 是索引
    never类型
      永不存在的值
      只能被never类型赋值
      是其他类型的子类型
    void:
      表示方法没有任何返回值
      function name():void(){}
